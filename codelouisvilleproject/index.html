About Me

My name is Autumn Keller, and I am creating this web page for my Code Louisville project. I am 32 years old and have a wonderful cat named Kramer. I also like to paint every now and then. I used to play the piano more and would like to start composing songs again. Finding a career where I can incorporate those hobbies, would be lovely. My plans are to continue learning new skills through Code Louisville as well as on my own.
